1.初始化仓库
```
git init
```

2.设置远程仓库
```
git remote add origin https://github.com/****/****.git
```

3.配置用户信息
```
git config user.email "****@gmail.com"
git config user.name "****"
```

4.直接创建并切换到 main 分支
```
git checkout -b main
```

5.查看本地状态
```
git status
git branch
```

6.拉取远程仓库内容
```
git pull origin main
```

7.如果有内容需要提交，先提交
```
git add output
git commit -m "更新内容"
```

8.推送到远程仓库
```
git push -u origin main
```
